# ZADANIE 5

Napisz program w C++, który tworzy dwa wątki. Pierwszy wątek ma wielokrotnie dodawać znak `'*'` do wspólnego łańcucha (typu `std::string`), natomiast drugi wątek ma usuwać kolejne znaki z tego łańcucha, dopóki nie będzie on pusty. Każdorazowe dodanie znaku powinno zostać zasygnalizowane wypisaniem litery `A` na ekran, a każdorazowe usunięcie znaku – wypisaniem litery `B`. Po zakończeniu działania obu wątków wypisz na końcu zawartość łańcucha, aby sprawdzić, co faktycznie pozostało po jednoczesnym dodawaniu i usuwaniu znaków.
