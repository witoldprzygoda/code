# ZADANIE 3

Napisz program w C++, który przyjmuje jako parametr zewnętrzny pojedynczą wartość całkowitą nieparzystą. Program powinien pobrać tę liczbę, a następnie narysować symetryczny trójkąt z gwiazdek. Na początku należy utworzyć napis zawierający liczbę gwiazdek równą wczytanej wartości. W każdej kolejnej linii należy zmniejszyć liczbę gwiazdek o dwa i zwiększyć liczbę wiodących spacji o jeden, aby uzyskać kształt trójkąta. Dla przykładowego parametru 5 końcowy wynik powinien wyglądać następująco:

```
*****
 ***
  *
```

Należy użyć obiektu typu `std::string` do przechowywania gwiazdek. Parametr podany zewnętrznie musi być liczbą dodatnią i nieparzystą. Jeżeli użytkownik nie poda parametru lub poda liczbę niepoprawną, program powinien wyświetlić komunikat o błędzie i zakończyć działanie.
