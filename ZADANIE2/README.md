# ZADANIE 2

Napisz program w C++, który zademonstruje użycie dowiązań strukturalnych w różnych sytuacjach. Zacznij od zdefiniowania tablicy obiektów typu `std::string` zawierającej trzy słowa i rozpakuj jej elementy do osobnych zmiennych przy pomocy dowiązań strukturalnych, po czym wypisz ich zawartość. Następnie stwórz obiekt typu `std::pair<std::string, std::string>`, wypełnij go przykładowymi danymi, ponownie użyj dowiązania strukturalnego do pobrania wartości i wyświetl je na ekranie. Kolejnym krokiem będzie obiekt typu `std::tuple<std::string, std::string, std::string>`, dla którego również należy wykonać analogiczne czynności. Na koniec zdefiniuj obiekt typu `std::array<std::string, 3>`, uzupełnij go danymi, rozpakuj zawartość do trzech zmiennych i wypisz je, wykorzystując taki sam mechanizm dowiązań strukturalnych. Dzięki temu zaprezentujesz, jak w C++17 można stosować rozpakowywanie elementów nie tylko z tablic, ale też z par, krotek oraz innych kontenerów.
